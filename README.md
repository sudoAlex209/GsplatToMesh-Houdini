These tools allow you to create polygonal meshes from gaussian splats in Houdini, and process roughness maps based on the spherical harmonics attributes.
They are not designed to be used in isolation, rather they are designed to be used in conjunction with other Houdini tools, as well as with RealityScan and Lichtfeld Studio.
